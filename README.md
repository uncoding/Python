#####切片
	切片主要是对数组进行一个切割的处理的功能，和PHP中的substr的功能基本相同
	具体的格式为: A[a:b:c]
	其中的参数为:
	A:当前需要处理的数组
	a:数组的起始位置 [可以省略、可以为负数]
	b:数组的结束位置 [可以省略、可以为负数]
	c:对应的步长 [可以省略、可以为负数]
	对应的测试文件为: test_substr.py
